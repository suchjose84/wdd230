Site Plan

Site Description

The purpose of this document is to describe the panahonweather.com website, it's intended audience, purpose, and design. This website will be used as a canvas to study the design principles of Web Fron-End Development in accordance to the instructions from WDD230 course of Brigham Young University-Idaho.

Domain Name

panahonweather.com

Site Name

The website will be called Panahon Weather. As the name describes, this will be an online location where people can check the current weather and weather forecasts.

Purpose Statement

The main purpose of Panahon Weather is to provide service to the public by helping them prepare for upcoming Typhoons and other weather related disasters. This website will show satellite images as well as weather reports and forecasts so that people can use it as a tool and help them with their everyday lives.
